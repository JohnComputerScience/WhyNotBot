# Sparky
Sparky is a robot meant to compete in the 2023 Robogames Autonomous Firefighter Competition


Wiring Rules:

Orange- direct from battery, 7.4v

Red- 6v line

Black- ground

White- digital signal/ A signal

Yellow- Analog Signal/ B signal
